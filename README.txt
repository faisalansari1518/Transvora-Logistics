TRANSVORA LOGISTICS — PROFESSIONAL WEBSITE V2
=============================================

FILES
- index.html               Main website
- styles.css               Design and responsive layout
- config.js                Edit business details here
- script.js                Navigation, animations and WhatsApp forms
- privacy.html             Basic privacy page
- terms.html               Basic terms page
- assets/logo-dark.svg     Header logo
- assets/logo-light.svg    Footer logo
- assets/favicon.svg       Browser icon

EDIT YOUR DETAILS
Open config.js and replace:
- Phone number
- WhatsApp number
- Email address
- City and state
- Business hours
- Social-media links

IMPORTANT
WhatsApp number must include country code but no +, spaces or dashes.
Example: 919876543210

GITHUB UPLOAD
1. Extract the ZIP.
2. Upload all files and the complete assets folder to your repository.
3. Keep index.html in the repository root.
4. GitHub > Settings > Pages.
5. Choose Deploy from a branch > main > /(root) > Save.

REPLACE IMAGES
The website uses professional online logistics images from Unsplash.
To use your own photos, replace the image URLs in index.html and styles.css.

BEFORE PUBLIC LAUNCH
- Replace all demo contact details in config.js.
- Add your actual registered legal name when available.
- Confirm every service listed is genuinely offered.
- Have the privacy policy and terms reviewed for your specific business.
- Test the website on phone and desktop.
